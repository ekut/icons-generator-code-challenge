# Product Overview

Icon Set Generator is a web application that generates consistent sets of 4 themed icons using AI image generation. Users provide a prompt (e.g., "Toys") and select from 5 preset visual styles to generate cohesive icon sets.

## Core Features

- Generate 4 unique, themed 512x512 PNG icons from a single prompt
- 5 preset visual styles for consistent aesthetics
- Optional brand color customization
- Downloadable PNG/JPG outputs
- Consistent visual style across all 4 icons in a set

## Key Requirements

- Icons must be different from each other (no duplicates)
- All icons in a set must share the same visual style
- Icons must be thematically related to the input prompt
- Uses Replicate API with FLUX-schnell model for generation
